🚀 HOE TE GEBRUIKEN:

1. Open je Roblox executor (Synapse X, KRNL, Fluxus, etc.)
2. Join een Roblox game
3. Attach/inject je executor
4. Kopieer de loadstring hierboven
5. Plak in je executor
6. Klik Execute
7. Wacht 2-3 seconden
8. Geniet van Inovo Productions! ✨

🔧 TROUBLESHOOTING:

❌ Script laadt niet?
   → Check of repository PUBLIC is
   → Wacht 1-2 minuten na uploaden
   → Check console (F9) voor errors

❌ UI niet zichtbaar?
   → Druk op K
   → Re-execute het script

❌ "404 Not Found"?
   → Check of alle bestanden geupload zijn
   → Wacht even (GitHub cache)
